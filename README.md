# OTDDA
OTDDA is a MATLAB-based optical-mechanical coupling simulation tool. Utilizing the Discrete Dipole Approximation (DDA) method, it enables fully coupled numerical simulation of electromagnetic, mechanical, and photothermal effects for arbitrarily shaped particles.
There are 7 codes in the file that can be run directly, namely:
# compaer_mie.m 
calculate the scattering field distribution at the target location and compare it with the Mie scattering results
# force.m
calculate the force curve in the set direction
# Optical_trap_stiffness.m
calculate the stiffness of the optical trap in the set direction
# photophoretic_force_position.m
calculate the photophoretic force curve in the set direction
# scattering_filed.m
calculate the scattering field distribution at the target location
# temperature_distribution.m
calculate the temperature distribution of spherical particles
# temperature_distribution_general.m
Calculate the temperature of non spherical particles at a slower rate than temperature_distribution.m
